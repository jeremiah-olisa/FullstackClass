Hello world this is our first HTML page

  • <!DOCTYPE> – Document type declaration
  • <html> – Root element
  • <head> – Head section
  • <title> – Document title
  • <meta> – Metadata
  • <link> – External resources
  • <body> – Main content
  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <br> – Line break
  • <hr> – Horizontal rule
  • <a> – Anchor (link)
  • <img> – Image
  • <video> – Video
  • <audio> – Audio
  • <ul> – Unordered list
  • <ol> – Ordered list
  • <li> – List item
  • <table> – Table
  • <tr> – Table row
  • <td> – Table cell
  • <th> – Table header cell
  • <thead> – Table head
  • <tbody> – Table body
  • <tfoot> – Table footer
  • <form> – Form
  • <input> – Input field
  • <button> – Button
  • <label> – Label for form elements
  • <select> – Drop-down list
  • <option> – Option in a drop-down
  • <textarea> – Multi-line text input
  • <div> – Generic container
  • <span> – Generic inline container
  • <strong> – Strong importance (bold)
  • <em> – Emphasis (italic)
  • <blockquote> – Block quotation
  • <code> – Inline code
  • <pre> – Preformatted text
  • <small> – Smaller text
  • <sup> – Superscript
  • <sub> – Subscript
